Commands:

(stackName).push(valToPush) -- Pushed new value to the top of the stack

(stackName).pop() -- removes and returns the top value from the stack

(stackName).peek() -- returns the top value from the stack

(stackName).isEmpty() -- returns whether the stack is empty

(stackName).isFull() -- returns whether the stack is full

(stackName).size() -- returns the amount of items in the stack
